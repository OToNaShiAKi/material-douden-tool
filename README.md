# Material DouDen Tool

采用 微拟物 FUI & Material 设计的 Bilibili 同传工具

任何使用问题、bug、功能需求可以提 [issue](https://github.com/OToNaShiAKi/material-douden-tool/issues)。

右侧 Release 中下载最新版即可，Windows 系统下载 exe 安装包，MacOS 系统下载 dmg 安装包，网络可稳定连接 GitHub 时软件支持自动更新。

---

## 基础操作

左侧选择同传标识符，可用 Tab 键快捷切换。

右侧为同传弹幕输入框，回车即可发送，Alt + 上下键可切换已发送弹幕，存在快捷短语时按下快捷键可立即发送短语。在同传输入框中按下 Alt+P 可立即打点。

当字符长度或发送频率超过 Bilibili 限制时，将自动延迟并切割弹幕，切割长度为当前直播间可发送最高字数。

## 主窗口功能

### 弹幕捕获

实时捕获当前选中直播间的弹幕与 Super Chat，同时获取历史十条弹幕与房间权限。

- 因界面大小原因一次只展示一个房间的弹幕捕获，联动情况可选择展示自己传的直播间
- 断线后自动重连
- 拥有房管权限时，可对捕获到的弹幕用户一键禁言
- 支持用户名、弹幕内容一键点击复制

### 选择房间

搜索并管理同传目标直播间。

- 搜索框自动展示关注列表正在直播的前 10 位
- 支持 UID、LiveID、昵称关键字搜索用户
- 支持添加、单选或多选房间，多选用于联动同传（弹幕发至多个直播间）
- 可配置直播间弹幕颜色和位置模式
- 可更换当前使用的粉丝牌，切换房间时支持自动换牌（联动时按选择顺序匹配第一个拥有的牌子，不持有则自动摘牌）

> 注：Bilibili 限制弹幕发送频率约 1 条/秒，多房间发送存在延迟。

### 歌词同传

从网易云音乐搜索歌曲歌词，支持自动轴播放和手动发送。

- 搜索结果为网易云共同检索并过滤掉无词无轴歌曲，支持按歌曲名、歌手、语言轴排序
- 点击歌曲切换到歌词页面，支持 +0.5s / -0.5s 调整歌词轴
- 重置播放即停止并重置歌词轴到起始位置；发送下句可立即发送高亮位置的下一句歌词
- 点击跳转按钮可从当前滚动位置处的歌词开始同传，歌词下方空白部分从属于该段歌词
- 可将搜索到的歌曲轴导出为 Aegisub 文件，部分歌曲支持自动 K 轴
- 左侧可选择歌词标识符（在标识符界面配置）

### 快捷短语

可配置 Alt+0~9、Ctrl+0~9 共 20 个快捷短语，在同传输入框中按下对应快捷键即可立即发送。

支持以下替换符：
- `{c}` — 输入框中的文本
- `{v}` — 剪切板文本
- `{s}` — 当前歌曲名
- `{a}` — 当前歌手名
- `{l}` — 当前播放位置歌词

### 标识符

配置同传弹幕的前缀和后缀，可按作用范围分为"同传"、"歌曲"、"全部"三种。

### 登录

支持 Bilibili 扫码登录，也可将浏览器中发送直播间弹幕时的请求 Cookie 粘贴登录。

- 登录成功后方可发送同传弹幕、选择房间等
- 实时监听 Cookie 状态（是否过期）

### 屏蔽词

设置屏蔽词及其替换文本，发送弹幕时自动替换。

- 添加/删除的词条同步到在线共享词库，可选择是否启用在线词库
- 被屏蔽的弹幕保存在安装目录下 `logs/forbidden-words.txt` 中，同时自动复制到剪贴板并提示
- 词库量大时可能存在处理耗时导致弹幕延迟，请谨慎选择

### 设置

- **主题色**：少女粉、宝石蓝、咖啡褐、咸蛋黄、夕阳红、早苗绿、罗兰紫共 7 种
- **自动抽红包**：自动抽取幸运红包，抽取时占用 1 条弹幕发送时间
- **自动换牌子**：切换同传房间时自动更换/摘下粉丝牌
- **自动复制被屏蔽弹幕**：被屏蔽弹幕自动复制到剪贴板
- **捕获过滤**：对弹幕捕获界面的弹幕进行过滤，被过滤弹幕不展示
- **在线词库**：开启/关闭在线共享屏蔽词库

## 辅助窗口功能

### 直播追帧

以低延迟、多线程对直播画面进行追帧播放。网络卡顿恢复后以 1.5 倍速追赶到当前帧（而非直接跳转），掉线过久则重连到最新帧。

- 可选择播放的直播间和画质（可选画质由主播直播画质和 Bilibili 推流决定）
- 在同传输入框中按下 Alt+P 即可在当前画面打点截图
- 截图自动叠加开播日期和已播时长水印，可在截图上输入关键字标注
- 支持批量导出所有截图

### 导出棉花糖

将棉花糖（Marshmallow）内容导出为图片卡片。

- 支持单独导出中文、翻译、合并样式
- 合并样式支持横排、竖排、反向横排和反向竖排四种布局
- 可直接在卡片中输入文本导出，也可传入 Excel 文件批量导出（第一列中文，第二列日文）
- 支持自定义卡片样式：容器颜色、背景颜色、文字颜色、提示颜色
- 支持自定义字体（从系统字体列表选择）和字号
- 支持上传自定义 Logo

### 禁言列表

查询并管理所有拥有房管权限的直播间中被禁言的用户。

- 自动查询房间界面中所有拥有房管权限的直播间的禁言列表
- 支持按 UID、昵称搜索用户并在指定直播间中禁言
- 可撤销已有的禁言

### 动态评论导出

将 Bilibili 动态帖子下评论中的视频推荐捕获并导出。

- 支持输入多个动态 ID（空格分隔），批量抓取
- 可筛选导出字段：封面、链接、标题、时长、推荐人等
- 支持导出为 Word（.docx）或 Excel（.xlsx）文档
- 可选择是否包含总时长统计
- 同时导出无视频链接的纯文字评论

### 歌词制作

创建、编辑和管理自制歌词，也可导入 Aegisub 文件。

- 支持手动输入歌词并设定起止时间轴
- 支持上传 Aegisub（.ass/.ssa）文件解析为歌词
- 支持双语歌词（原文 + 翻译）
- 支持 K 轴（卡拉 OK 逐字时间轴）编辑
- 自制歌词可保存到本地，用于歌曲同传
- 可将歌词导出为 Aegisub 文件
